# Guía Turística - América del Sur 🌎

Sitio web de guía turística sobre destinos de América del Sur,
desarrollado como proyecto personal para consolidar habilidades
de desarrollo Front End.

## 🚧 Estado: En desarrollo

Página principal completada. Páginas individuales de destinos
y países en progreso.

## 🛠️ Tecnologías

- React
- JavaScript
- HTML5
- CSS3

## 📄 Página principal (Index)

- **Destinos destacados** — grid con 6 lugares de América del Sur,
  cada uno con su página dedicada (en desarrollo)
- **Países de América del Sur** — sección con todos los países,
  cada uno con su propia página
- **Prepara tu viaje** — links directos a plataformas externas
  para reservar alojamiento, vuelos y paquetes turísticos
